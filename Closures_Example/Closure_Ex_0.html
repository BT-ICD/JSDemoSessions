Returning Functions

>

Functions are objects, so they can be used as return values. This means that a function doesn’t need to return some sort of data value or array of data as a result of its execution. A function can return another more specialized function, or it can create another function on-demand, depending on some inputs. Here’s a simple example: A function does some work, probably some one-off initialization, and then works on its return value.
