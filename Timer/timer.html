Timer demo
  • setTimeout() - The setTimeout() method sets a timer which executes a function or specified piece of code once the timer expires.
  • setInterval() - The setInterval() method, repeatedly calls a function or executes a code snippet, with a fixed time delay between each call. It returns an interval ID which uniquely identifies the interval, so you can remove it later by calling clearInterval().
  • clearInterval()- The clearInterval() method cancels a timer, repeating action which was previously established by a call to setInterval().
