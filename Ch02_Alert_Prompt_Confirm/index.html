Functions to interact with user:

  • alert- alert(message)
  • prompt - result = prompt(title, [default]);
  • confirm - result = confirm(question);

alert

It provides a message in a window to the user with OK button.

prompt

It allows user to input string. Dialog contains two buttons OK and Cancel. When User press Cancel it return null.

confirm

It provides functionality to ask user a question with two buttons. Ok and Cancel Return result as true if Ok is selected otherwise false
